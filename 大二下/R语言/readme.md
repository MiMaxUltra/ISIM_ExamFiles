### R语言

前置课程：应用统计学（最好有统计基础）

主要讲了如何利用R语言进行各种金融上的数据统计分析，如果未来是走金融科技方向，可以仔细学习。本课程内的所有例程，都是可以根据场景抽出来改编后就可以适配的，十分的珍贵/doge

**使用前需要下载R Studio（要不然没有的library不好安装，R Studio可以直接点击安装），会比折腾VS Code好很多，如果打开时乱码，请分别尝试用utf-8和gb-2132两种编码格式打开，总有一款适合你。**
